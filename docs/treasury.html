treasury: US Treasury XML Feed Wrapper

Description

Download daily interest rates from the US Treasury XML feed. Leveraging https://home.treasury.gov/treasury-daily-interest-rate-xml-feed, this package serves as a wrapper, facilitating the retrieval of daily treasury rates across various categories, including par yield curves, treasury bills, long-term rates, and real yield curves.

Return the daily treasury bill rates

Description

Return the daily treasury bill rates

Usage

tr_bill_rates(date = NULL)

Arguments

date

character(1) or numeric(1) date in format yyyy or yyyymm. If NULL, all data is returned. Default NULL.

Value

A data.frame() with columns date, type, maturity and value or NULL when no entries were found.

References

https://home.treasury.gov/treasury-daily-interest-rate-xml-feed

See Also

Other treasury data: tr_long_term_rate(), tr_real_long_term(), tr_real_yield_curve(), tr_yield_curve()

Examples


# get data for a single month
tr_bill_rates("202201")
# or for the entire year
tr_bill_rates(2022)
