Launch 'ANOVAShiny2' Interface

Description

ANOVAShiny2() loads interactive user interface built using R 'shiny'.

Usage

ANOVAShiny2()

Details

Start ANOVAShiny2

The interactive user interface is to provide an easy way for anova analysis and downloading relevant plots.

Value

Nothing

Examples

if(interactive()){
library(rmarkdown)
ANOVAShiny2()
}
