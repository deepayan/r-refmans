Type: Package
Title: Analyse Open-Ended Survey Responses in Finnish
Version: 1.0.0
Description: Annotates Finnish textual survey responses into CoNLL-U format using Finnish treebanks from https://universaldependencies.org/format.html using UDPipe as described in Straka and Strakov√° (2017) <doi:10.18653/v1/K17-3009>. Formatted data is then analysed using single or comparison n-gram plots, wordclouds, summary tables and Concept Network plots. The Concept Network plots use the TextRank algorithm as outlined in Mihalcea, Rada & Tarau, Paul (2004) https://aclanthology.org/W04-3252/.
