Launch 'STAT2' Interface

Description

STAT2() loads interactive user interface built using R 'shiny'.

Usage

STAT2()

Details

Start STAT2

The interactive user interface is to provide an easy way for basic statistical analysis and downloading plots.

Value

Nothing

Examples

if(interactive()){
library(rmarkdown)
STAT2()
}
