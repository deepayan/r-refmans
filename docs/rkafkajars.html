External Jars Required for Package 'rkafka'

Description

The 'rkafkajars' package collects all the external jars required for the 'rkafka' package. These external jars are quite large in size (12MB) and have a slow release cycle. By separating the Java and R development, the storage footprint on CRAN is reduced. By using the 'rJava' package that links R and Java, we can use the excellent work already done by the folks and provide the functionality of using 'KAFKA' in R
