Launch 'STAT' Interface

Description

STAT() loads interactive user interface built using R 'shiny'.

Usage

STAT()

Details

Start STAT

The interactive user interface is to provide an easy way for basic statistical analysis and downloading plots.

Value

Nothing

Examples

if(interactive()){
library(rmarkdown)
STAT()
}
