Arguments

x

[numeric(1)]
The estimate.

eif

[numeric(n)]
The influence function.

weights

[numeric(n)]
Optional sampling weights.

id

[character(n)]
Optional cluster identifiers.

critical_value

[numeric(1)]
Optional critical value for constructing confidence interval.

Value

An 'S7' object of class influence_func_estimate.
