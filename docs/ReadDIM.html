Import ESA SNAP .dim file

Usage

read.dim(dim_path, noDataV = NA, stacked = FALSE)

Arguments

dim_path

Path to the .dim file.

noDataV

Value of no data used in raster.

stacked

If true create a stacked image else list

Author(s)

Subhadip Datta

Examples

library(raster)
library(ReadDIM)
im<-system.file("exd","td.dim",package = "ReadDIM")
read.dim(im)
