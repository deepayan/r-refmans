bundesligR

Description

All final tables of Germany's highest football (soccer) league, the Bundesliga, from 1964 to 2016.

Usage

bundesligR

Format

A data frame with twelve variables:

Season

The year the respective season started. The year of the final table is Season + 1.

Position

Team's position in the final table.

Team

Team name.

Played

Number of games played.

W

Number of games won.

D

Number of draws.

L

Number of games lost.

GF

Goals for the respective team.

GA

Goals against the respective team.

GD

Goal difference.

Points

Achieved points.

Pts_pre_95

Achieved points according to the pre 1995 two-point-system.
