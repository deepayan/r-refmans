Launch 'VTShiny' Interface

Description

VTShiny() loads interactive user interface built using R 'shiny'.

Usage

VTShiny()

Details

Start VTShiny

The interactive user interface is to provide an easy way for variance analysis.

Value

Nothing

Examples

library(rmarkdown)
VTShiny()
}
