Title: Irish Weather Observing Stations Hourly Records for 2017
Version: 0.0.2
Description: Named after the Irish name for weather, this package contains tidied data from the Irish Meteorological Service's hourly observations for 2017. In all, the data sets include observations from 25 weather stations, and also latitude and longitude coordinates for each weather station. Now includes energy generation data for Ireland and Northern Ireland (2017), including Wind Generation data.
