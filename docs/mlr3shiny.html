Title Launch mlr3shiny

Description

Launch an instance of mlr3shiny to perform machine learning in Shiny using mlr3.

Usage

launchMlr3Shiny(test = FALSE)

Arguments

test

Argument used for testing purposes. Default is FALSE.

Value

None

Examples

if (interactive()) {
   launchMlr3Shiny()
}
