Results of touch type experiment with two subject using the PlatoWork tDCS headset for possible improvement of learning speed.

Usage

plato

Format

A data frame with 225 rows and 6 variables:

date

time of test

subject

name of person doing the typing test

wpm

words per minute in typing test

error

error rate in typing test

stimulus

type of stimulus

session

session id, unique for each subject
