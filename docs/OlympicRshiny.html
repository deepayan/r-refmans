Olympic data

Description

Data of Olympic games from 1896 to 2016 are stored here, which is available at Kaggle.

Usage

Olympic

Format

A data frame with

Name

Athlete's name

Sex

Male or Female

Age

Age of the athlete in integer form

Height

In centimeters

Weight

In kilograms

Team

Team name

NOC

National Olympic Committee 3-letter code converted into region

Games

Year and season

Year

Year the olympic games was held

Season

Summer or Winter olympic games

City

Host city

Sport

Sport

Event

Event

Medal

Gold, Silver, Bronze, or No medal

Examples

plot(Olympic$Height,Olympic$Weight)
