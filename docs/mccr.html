Calculate the Matthews correlation coefficient (MCC) score

Description

Calculate the Matthews correlation coefficient (MCC) score

Usage

mccr(act, pred)

Arguments

act

actual values (vector), 1 (positive), or 0 (negative)

pred

predict values (vector), 1 (positive), or 0 (negative)

Value

MCC score

Examples

set.seed(18)
act <- abs(round(rnorm(100))) %% 2
pred <- abs(round(rnorm(100))) %% 2
mccr(act, pred)
