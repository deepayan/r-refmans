Photogrammetric Distances Measurer

Description

This package can measures real distances in pictures. With PDM() function, you can choose one .jpg file, select the measure in mm of scale, starting and and finishing point in the graphical scale, the name of the measure, and starting and and finishing point of the measures. After, ask the user for a new measure.
