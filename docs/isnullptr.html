Check if an 'externalptr' is a null pointer

Description

isnullptr checks if an object of type 'externalptr' is a null pointer.

Usage

isnullptr(pointer)

Arguments

pointer

object of type 'externalptr'

Value

TRUE if null pointer, FALSE otherwise

Examples

isnullptr(new("externalptr"))
isnullptr("any_data")
