Add a package

Description

Add a package

Usage

add(packages, remotes = c())

Arguments

packages

Packages to add

remotes

Remotes to add

Value

No return value

Examples

## Not run: 

jetpack::add("randomForest")

jetpack::add(c("randomForest", "DBI"))

jetpack::add("DBI@1.0.0")

jetpack::add("plyr", remote="hadley/plyr")

jetpack::add("plyr", remote="local::/path/to/plyr")

## End(Not run)
