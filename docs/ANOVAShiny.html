Launch 'ANOVAShiny' Interface

Description

ANOVAShiny() loads interactive user interface built using R 'shiny'.

Usage

ANOVAShiny()

Details

Start ANOVAShiny

The interactive user interface is to provide an easy way for anova analysis and downloading relevant plots.

Value

Nothing

Examples

if(interactive()){
library(rmarkdown)
ANOVAShiny()
}
