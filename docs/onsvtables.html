Add styles to a {gt} table.

Description

tab_onsv() uses ⁠gt::tab_*()⁠ functions to customize a gt_tbl object.

Usage

tab_onsv(data)

Arguments

data

A gt_tbl object.

Value

A gt_tbl customized with tab_style() and tab_options().

Examples

df <- data.frame(
  var1 = c("lorem", "ipsum", "dolor"),
  var2 = c(1, 2, 3)
)

tbl_onsv <-
  df |>
  gt::gt() |>
  tab_onsv()
