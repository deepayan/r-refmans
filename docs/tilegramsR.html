sf_Datamap.io.tilegram

Description

A 'sf' object where each polygon represents a state

A 'sf' object of centroids of each state

Usage

sf_Datamap.io.tilegram

sf_Datamap.io.tilegram.centers

Format

sf

Examples

## Not run: 
library(leaflet)
library(tilegramsR)
data <- sf_Datamap.io.tilegram
leaflet(data,
  options=leafletOptions(crs=leafletCRS("L.CRS.Simple"))) %>%
  addPolygons()

## End(Not run)

tilegramsR

Description

Tilegrams are tiled maps where the region size is proportional to the certain characteristics of the dataset. This package provides several such tilegrams as simple feature (sf) objects.
